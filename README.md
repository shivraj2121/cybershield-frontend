# CyberShield — AI Cybersecurity Web App

A full-stack AI-powered cybersecurity web application that provides intelligent security tools.

## Features
- 🤖 AI Chat Assistant (cybersecurity, coding, general knowledge)
- 🔐 Password Strength Checker
- 🛡️ Secure Login & Registration System
- 🌐 Modern dark cyber-themed UI

## Tech Stack
- React.js
- CSS3
- Axios
- React Router DOM
- React Icons

## Setup & Run
1. Clone the repository
   git clone https://github.com/shivraj2121/cybershield-frontend.git
2. Install dependencies
   npm install
3. Run the app
   npm run dev
4. Open http://localhost:5173
